## 准备工作

开始之前，请确保您具备以下条件：
- 拥有一个活跃的 LooPIN 账号。
- 确保您有足够的空闲 GPU 实例在线，且未已经加入到流动性池中，若您打算出售 GPU 使用时间。
- 如果您想购买 GPU 使用时间，确保您有足够的 $LOOPIN 余额。

## 如何购买 GPU 使用时间

按以下步骤使用 $LOOPIN 代币购买 GPU 使用时间：

1. **前往流动池 Pool**：从这里开始您的购买。
2. **选择 $LOOPIN 代币数量**：决定您希望投入多少 $LOOPIN 代币来购买 GPU 使用时间。
3. **选择 GPU**：使用滑动条选择您希望同时使用的 GPU 数量。系统会自动为您的交换计算每个 GPU 的有效使用时间，至少购买 1 小时。
4. **确认交易详情**：检查即将用于交易 GPU 使用时间的 $LOOPIN 代币量。
5. **执行交换**：点击交换并成功交易后，前往 **Server** 界面下的 **Rent 租用服务器**，从您新租用的远程服务器访问 Jupyter 笔记本。通常，启动一个实例需要 2-4 分钟。

## 如何出售 GPU 使用时间

若您希望出售您的 GPU 空闲时间，可按以下步骤操作：

1. **进入流动池 Pool**：您可以在此列出希望出售的 GPU 使用时间。
2. **设定可售时间**：选择您想要出售的 GPU 使用时间数量。例如，如果您的 1 个 GPU 正在在线，您最多可以出售 24 小时，这将显示为您的 GPU 时间余额。您可以切换不同的 GPU 来调整您的出售额。
3. **选择目标 GPU 并确认交易详情**：选择您希望分配的 GPU，并确认交换的具体详情，包括交换的金额。
4. **需要抵押**：出售 GPU 使用时间需要预先支付抵押金，此金额略高于您将获得的收益。如无任何错误完成 GPU 使用时间提供后（网络稳定，机器正常运行），抵押金将退还；否则，将作为罚金没收。并被永久销毁以保持代币平衡。更多抵押信息请查阅我们的代币文档。
5. **完成交换**：交换成功后，检查您的服务器状态，您的 GPU 将显示为 **In used 正在使用**。同时，前往钱包部分核实您的 $LOOPIN 余额。

遵循这些步骤，无论您是希望购入新的GPU计算资源还是希望通过您的闲置 GPU 获利，都可以在 LooPIN 平台上有效地交易 GPU 使用时间。
