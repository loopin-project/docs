## 开始之前

在开始之前，请确保你的硬件和软件满足以下要求：

- 注册并拥有 [loopin.network](https://loopin.network) 的账户。如果你还没有账户，现在就去注册一个吧！
- 确保你的 **Nvidia GPU** 正在运行并已连接到 LooPIN 网络。

## 开始添加流动性

一切准备就绪后，就可以开始利用你的闲置 GPU 时间了：
- 在导航栏中点击 Pool，进入流动性管理界面。
- 选择 GPUs，然后从你的闲置 GPU 中挑选用于添加流动性。
- 确保你的钱包余额充足，因为实际抵押的 $LOOPIN 代币将从你的钱包中扣除，并在你取回流动性时返还。
- 添加流动性后，返回 Server 界面。你将看到你的服务器已被标记为 LP，表示它已加入流动性池。

## 如何取回流动性

- 前往 Server 页面，选择您想要操作的 GPU。
- 点击 Remove Liquidity 按钮。
- 操作完成！您的服务器将不再属于流动性池，您可以自由出售 GPU 使用时间或移除服务器。

### 设置完成！

恭喜您！您的服务器现已加入 LooPIN 的流动性池，并开始享受每日的质押奖励。
