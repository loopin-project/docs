# 硬件要求

## 加入LooPIN网络的基本条件

想要加入LooPIN网络，您的系统需要符合以下最低标准：

1. **GPU**：至少配置一块Nvidia GPU。
2. **操作系统**：需要运行Ubuntu或Debian的Linux系统。
3. **网络**：必须有一个公网IP地址。
4. **端口**：需确保所有端口对LooPIN网络开放。
5. **RAM**：每块GPU至少需要4GB内存。
6. **磁盘空间**：至少100GB。
7. **Nvidia驱动程序**：版本需达到525.00或以上。
8. **CUDA版本**：需12.0或以上。

## 网络配置要求

LooPIN网络基于Kubernetes (K8S)运行，以下是必须满足的网络端口要求：

| 协议 | 方向 | 端口范围    | 用途         | 应用对象         |
| ---- | ---- | ----------- | ------------ | ---------------- |
| TCP  | 入站 | 22       | LooPIN连接  | 全部设备   |
| TCP  | 入站 | 9988     | LooPIN服务  | 全部设备   |
| TCP  | 入站 | 10250       | Kubelet API  | 自身及控制平面   |
| TCP  | 入站 | 10256       | kube-proxy   | 自身及负载均衡器 |
| TCP  | 入站 | 30000-32767 | NodePort服务 | 全部设备         |

## 安装流程

安装时，请在终端中执行以下命令（详细步骤见[快速上手](/docs/quick-start/onboard)）：

```bash
curl -fsSL https://files.loopin.network/install.sh | sudo bash -s -- --code <your-unique-code>
```
