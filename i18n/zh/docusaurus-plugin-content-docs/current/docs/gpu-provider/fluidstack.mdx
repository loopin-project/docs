# FluidStack

## FluidStack是什么

[FluidStack](https://console.fluidstack.io) 提供AI专属GPU及大语言模型（LLM）训练服务。您可以访问位于全球各地数据中心的超过50000台GPU，包括H100和A100型号。

## 如何在Lambda Cloud注册并创建新实例

使用FluidStack非常简单，只需遵循以下几个步骤即可开始。

### 创建新实例

1. **开始创建实例:**
- 在控制台左侧点击“虚拟机”，开始新实例的设置。
2. **选择操作系统模板:**
- 选择“Ubuntu 22.04 (Plain)”作为操作系统。
3. **选择实例类型:**
- 选择A100 PCIE 80GB型号的GPU实例，该型号为高需求应用提供卓越性能。
4. **设置凭证:**
- 若您还未创建SSH密钥，可选择“添加新的SSH密钥”，并按提示步骤进行创建与注册。
5. **部署实例:**
- 点击“部署”，开始实例的创建过程。

6. **监控实例设置:**
- 等待实例完全启动，并记录下显示的IP地址。

### 连接到您的实例

7. **初始化并连接到您的实例:**
- 确认您的实例已激活并获得IP地址。
- 通过以下命令在终端建立一个安全的SSH连接：
```
ssh ubuntu@<your-ip>
   ```
- 这条命令将建立一个安全的连接到您的实例，使您可以开始远程管理。

## 将您的FluidStack实例连接到LooPIN网络

## 准备工作

在开始之前，请确保您满足以下条件：

- 保持网络连接稳定，并确保端口开放，以便访问集群。
- 您需要一台安装了 Nvidia GPU 的 Ubuntu 系统，目前 LooPIN 仅支持此配置。

## 上手指南

请跟随我们的 互动式教程 进行快速且简单的设置。

### 如何加入 LooPIN

1. **注册 LooPIN**: 访问 [loopin.network](https://loopin.network)，使用您的电子邮件或 Google 账户进行注册。
2. **配置您的设备**: 确保您有一台装有 Nvidia GPU 的裸机。在导航栏中找到“服务器”一栏，点击“部署新机器”。
3. **命令行界面工作注册**:
 - 在 Ubuntu 系统中打开终端。
 - 执行以下命令，将您的设备注册为 LooPIN 工作节点（具体命令参见指导页面）：
```bash
curl -fsSL https://files.loopin.network/install.sh | sudo bash -s -- --code <your-unique-code>
```
 - 此命令将下载并执行安装脚本，用您的唯一代码注册设备。
4. **安装及验证**: 脚本会检测设备是否符合要求，并安装必要的驱动程序，包括 Nvidia GPU 驱动。完成后会进行设备验证并完成注册。安装成功后，您将看到以下提示：
```bash
LooPIN Client Connected! Waiting for the on-chain verification...
```
这表示客户端程序已安装并在等待网络连接。首次连接大约需要5-10分钟，具体时间取决于网络速度。

5. **接入集群**: 一旦您的设备成功连接到我们的集群并显示在线状态，您便可以开始利用您的空闲 GPU 时间参与到共享池中或进行流动性交换。

## 安装需要注意

为确保安装顺利进行，请准备：

- **稳定的网络连接**: 以便下载所需文件及与 LooPIN 网络进行通信。
- **装有 Nvidia GPU 的 Ubuntu 系统**: 这一点对于确保硬件与 LooPIN 的兼容性至关重要。

按照这些步骤操作，您将能够充分利用您的硬件资源，在 LooPIN 网络上发挥作用。
